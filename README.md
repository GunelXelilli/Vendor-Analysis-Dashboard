# 📊 Vendor Analysis Dashboard

## 🚀 Project Overview
This project focuses on analyzing vendor performance, sales trends, profitability, and inventory efficiency using **Python, SQL, and Power BI**.  
The dashboard provides interactive insights to support data-driven business decisions and optimize vendor management strategies.

---

## 🎯 Key Objectives
✔ Evaluate vendor contribution to total sales  
✔ Identify top-performing and low-performing vendors  
✔ Analyze revenue trends over time  
✔ Improve inventory and purchasing decisions  
✔ Detect business opportunities through data insights

---

## 📌 Dashboard Features

### 📈 Performance Metrics
- Total Sales  
- Total Profit  
- Profit Margin %  
- Number of Vendors  
- Average Order Value  

### 🏆 Vendor Insights
- Top Vendors by Revenue  
- Lowest Performing Vendors  
- Vendor Ranking Analysis  

### 📦 Inventory Analysis
- Stock Efficiency  
- ABC Classification  
- Product Contribution Analysis  

### 📅 Time-Based Analysis
- Monthly Sales Trends  
- Seasonal Patterns  
- Growth Rate Comparison  

---

## 🛠 Tools & Technologies
- 🐍 Python (Pandas, NumPy, Matplotlib, Seaborn)
- 🗄 SQL
- 📊 Power BI
- 📒 Google Colab

---

## 📚 Analytical Methods Used
- ABC Analysis  
- Pareto Analysis (80/20 Rule)  
- Cohort Analysis  
- Hypothesis Testing  
- KPI Performance Tracking
